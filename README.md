# Sales-performance-Dashboard
This project presents an interactive Sales Performance &amp; Profit Analysis Dashboard built using Power BI to uncover key business insights from sales data. The dashboard helps analyze sales trends, profit distribution, customer behavior, and regional performance.

 # Project Overview

The objective of this project is to transform raw sales data into meaningful insights that support data-driven decision-making. The dashboard provides a comprehensive view of sales, profit, quantity, and category-level performance across different regions and shipping modes.

 # Key Features

✅ KPI Metrics

Total Sales: 2.35M

Total Profit: 283K

✅ Sales Analysis

Sales distribution by Ship Mode (Economy, Economy Plus, Priority, Immediate)

Sales performance across different Countries

✅ Category Insights

Quantity distribution by Category (Furniture, Office Supplies, Technology)

Sales contribution by Sub-Category (Phones, Chairs, Storage, etc.)

✅ Geographical Analysis

Map visualization showing sales distribution by State and Country

✅ Comparative Analysis

Category-wise comparison of sales and order count

Economy shipping mode contributes the highest sales

Certain sub-categories like Phones & Copiers generate high revenue
